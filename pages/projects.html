hoptions

Removing barriers to the very best craft beer using mobile image recognition technology

React Native | GraphQL | Viro AR

Hoptions is a solution developed to address the problem of decision fatigue in the experience of purchasing and consuming craft beer. User research identified the emotions experienced by the consumer: overwhelming choice, a desire to try something, before settling for a known beer.

Hoptions' recommendation algorithm and image recognition technology finds new beers based on user preferences. Users can filter recommendations by brewery, category - even hops type. Share user profile with friends to build the perfect round of beers. See the MVP in action here.


seemless

Disrupting inefficiencies within the fashion industry for a more sustainable future

Brand | Concept | UI/UX Design

seemless is a proof-of-concept project addressing overproduction in the fashion industry [reduce, reuse, recycle]. Research suggests that the profitable production of fabric requires over-commitment from either consumer or producer in order to reach minimum production runs.

By crowdsourcing fabric speculative production is reduced. Independent designers can pool resource to access economies of scale and small producers can access a large consumer market for sustainable products. View the value proposition landing page here.


spotify

Exploring recommendation algorithms in Spotify from an opiniated users perspective

API | Cluster Analysis | Design Thinking

An investigation into Spotify recommendations using the statistical analysis technique k-means clustering. My hypothesis posits there is a heavy weighting toward popularity in the app which results in a failure to handle artist complexity and disappoints a defined user persona.

Using the Spotify API, I found that my own playlists were much more sonically consistent: similarity within tracks rather than artists. I used the findings to propose a feature whereby the user can adjust the recommendation engine. Find out more about the proposal here.
